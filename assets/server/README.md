# Server Binaries

Place the llama-server binary for your platform in this directory.

## Download

Download pre-built binaries from the [llama.cpp releases](https://github.com/ggerganov/llama.cpp/releases).

## Platform Binaries

| Platform | Binary Name |
|----------|-------------|
| Linux x86_64 | `llama-server` |
| macOS ARM64 | `llama-server` |
| Windows x86_64 | `llama-server.exe` |

The application will automatically detect and use the binary from this directory.
